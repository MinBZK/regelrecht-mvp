# Wet open overheid (Woo) — Complete Legislative History

Kamerstuk 33328 (original) + 35112 (novelle)
BWBR0045754

## Summary

- **Duration**: 9+ years (5 juli 2012 → 1 mei 2022 inwerkingtreding)
- **Type**: Initiatiefwetsvoorstel (private member's bill)
- **Initiator**: Mariko Peters (GroenLinks)
- **6 sponsorship handovers**: Peters → Voortman → Schouw/Verhoeven/Van Weyenberg → Snels/Sneller
- **Novelle**: Yes (35112) — rare procedural instrument
- **Replaced**: Wet openbaarheid van bestuur (Wob, 1991)
- **Catalyst**: Toeslagenaffaire shifted political support

## Key Narrative Moments for Visualization

1. **2012**: External contributor opens a PR (initiatiefwet by lone MP)
2. **2012-2015**: Slow build — RvS review, committee work, MP handovers (the PR keeps getting reassigned)
3. **April 2016**: Intense burst — 4 nota's, 9 amendments, debate + vote in 3 weeks
4. **2016-2017**: Government deliberately stalls with impact analysis ("your code is unexecutable")
5. **2019**: The novelle compromise — a dependent fix-PR to address concerns
6. **2020-2021**: Novelle goes through TK; previously-rejected amendments now pass (political shift)
7. **Oct 2021**: Both PRs merged simultaneously in Eerste Kamer
8. **May 2022**: Deployed to production
