Hoe het werkt

De CSS-only tabs gebruiken hidden radio inputs gecombineerd met de :has() selector.

/* Hide radio inputs */ .tabs__input { position: absolute; opacity: 0; pointer-events: none; } /* Active State - Using :has() for modern browsers */ .tabs:has(#tab-1:checked) .tabs__label[for="tab-1"] { background-color: var(--color-primary); color: var(--color-white); } /* Show active panel */ #tab-1:checked ~ .tabs__panels .tabs__panel:nth-child(1) { display: block; }
