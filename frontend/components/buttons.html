Buttons

Button varianten voor acties en navigatie.

Overzicht Buttons List Tabs Forms Navigation

Primary Button

Gebruik voor hoofdacties zoals opslaan, bewerken of bevestigen.

<button class="btn btn--primary">Primary Button</button> <button class="btn btn--primary"> <svg class="btn__icon">...</svg> Bewerken </button>

Secondary Button

Gebruik voor secundaire acties zoals annuleren of delen.

<button class="btn btn--secondary">Secondary Button</button> <button class="btn btn--secondary"> <svg class="btn__icon">...</svg> Deel </button>

Ghost Button

Gebruik voor subtiele acties in lijsten of toolbars.

<button class="btn btn--sm btn--ghost">Bewerk</button> <button class="btn btn--sm btn--secondary">+ Nieuwe</button>

Toolbar Buttons

Icon-only buttons voor toolbars en editors.

<button class="editor-toolbar__btn" title="Bold"> <svg class="editor-toolbar__icon">...</svg> </button> <!-- Active state --> <button class="editor-toolbar__btn editor-toolbar__btn--active"> ... </button>

Lid Selector Buttons

Pill-style buttons voor selectie in de machine-editor.

<div class="lid-selector"> <button class="lid-selector__btn lid-selector__btn--active">Lid 1</button> <button class="lid-selector__btn">Lid 2</button> <button class="lid-selector__btn">Lid 3</button> </div>
